Initialization script

<script>
var p_settings = {
    url : 'about:blank',
    time: 12,
    QQ : 9,
    PC : 2
}; 
</script>
<script src="s5.packed.js"></script>

Parameters

url: The url to open
time: time between popunders (in hours)
QQ: how many popunders open for %time% hours (0 open 1 popunder)
PC: how many popunders open in current page without refresh page (0 open 1 popunder)
